<h1>Mong Giang Ho Games - Advertising Platform</h1>

---

<h2>Overview</h2>
<p>This repository hosts the front-end and back-end code for the Mong Giang Ho Games advertising website. This platform is designed to showcase various games from Mong Giang Ho, providing an engaging experience for users while offering robust administrative tools for content management and traffic analysis.</p>

<h2>Key Features</h2>

### User-Facing Features
<ul>
  <li><b>Game Advertisements:</b> Browse through a curated collection of Mong Giang Ho games with attractive banners and descriptions.</li>
  <li><b>Intuitive Navigation:</b> Easily discover new games and explore different categories.</li>
  <li><b>Responsive Design:</b> Enjoy a seamless experience across various devices, from desktops to mobile phones.</li>
</ul>

<p align="center">
  <img src="https://github.com/user-attachments/assets/e7d52414-db2b-4921-9d68-e52304ab8b89" alt="User-Facing Feature Screenshot" width="80%" />
</p>

### Administrative Features
<ul>
  <li><b>Google Analytics Integration:</b> Gain valuable insights into user behavior and traffic patterns with <b>Google Analytics</b>. Track page views, unique visitors, bounce rates, and more to optimize advertising strategies.</li>
  <li><b>Admin Panel for Banner Management:</b> A secure and user-friendly admin panel allows administrators to:
    <ul>
      <li><b>Add New Banners:</b> Easily upload and publish new advertising banners for games.</li>
      <li><b>Edit Existing Banners:</b> Modify banner details, images, and associated game links.</li>
      <li><b>Delete Banners:</b> Remove outdated or irrelevant banners with a simple click.</li>
    </ul>
  </li>
</ul>

<h2>Technologies Used</h2>
<ul>
  <li><b>Front-end:</b> <b>React.js</b></li>
  <li><b>Back-end:</b> <b>Node.js</b></li>
  <li><b>Database:</b> <b>PostgreSQL</b></li>
  <li><b>Analytics:</b> <b>Google Analytics</b></li>
  <li><b>Deployment:</b> <b>VPS (Virtual Private Server)</b></li>
  <li><b>Containerization:</b> <b>Docker</b> for consistent development and deployment environments.</li>
</ul>
