<h1>Mong Giang Ho Games - Advertising Platform</h1>

<h2>Overview</h2>
<p>This repository hosts the front-end and back-end code for the Mong Giang Ho Games advertising website. This platform is designed to showcase various games from Mong Giang Ho, providing an engaging experience for users while offering robust administrative tools for content management and traffic analysis.</p>

<h2>Key Features</h2>
<h3>User-Facing Features</h3>
<p>Game Advertisements: Browse through a curated collection of Mong Giang Ho games with attractive banners and descriptions.

Intuitive Navigation: Easily discover new games and explore different categories.

Responsive Design: Enjoy a seamless experience across various devices, from desktops to mobile phones.</p>

<h3>Technologies Used</h3>
<p>Front-end: React.js

Back-end: Node.js

Database:PostgreSQL

Analytics: Google Analytics

Deployment: VPS

Docker: Containerization for consistent development and deployment environments.</p>

<h3>How to Use</h3>
<p>Simply visit the website to start exploring. You can use the navigation bar to switch between the Heroes, Treasures, and Spirit Beasts sections. Utilize the search and filter functionalities to refine your results. Link: http://103.92.25.7:8081/</p>
